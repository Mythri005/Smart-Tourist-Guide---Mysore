# 🌍 Smart Tourist Guide – Mysore

## 📌 Overview

Smart Tourist Guide – Mysore is a web application designed to help users explore popular tourist destinations in and around Mysore. The platform provides detailed information about places, along with nearby hotels and restaurants, making travel planning easier and more convenient.

---

## ✨ Features

### 🏠 Dashboard (Landing Page)

* Attractive landing page introducing Mysore tourism
* Search functionality to quickly find places
* Category-based exploration (Historical, Nature, Religious, Attractions)
* Top Recommended Places section

### 📍 Explore Places

* View a curated list of tourist destinations
* Filter places based on categories
* Each place displayed with image, rating, and location

### 📄 Place Details

* Detailed information about each place
* Timings, entry fee, and description
* Direct options to:

  * View location on map
  * Get directions
  * Search on Google

### 🏨 Nearby Hotels

* Displays hotels near selected place
* Redirects to Google for price and booking details

### 🍴 Nearby Restaurants

* Displays restaurants near selected place
* Redirects to Google for menu, reviews, and ratings

### ❤️ Favorites

* Users can save their favorite places
* Easy access to shortlisted destinations

### 🧭 Planner

* Plan visits by selecting places
* Helps organize travel itinerary

---

## 🛠️ Technologies Used

### Frontend

* React.js
* CSS (custom styling)

### Backend

* Node.js
* Express.js

### Database

* SQLite

---

## 🧠 Key Concepts Implemented

* Component-based architecture
* REST API integration
* Dynamic data rendering
* State management using React Hooks
* Relational mapping between places, hotels, and restaurants
* External integration with Google for maps, reviews, and directions

---

## 🎯 Design Approach

The application is designed with a focus on:

* Simplicity and usability
* Clean and responsive UI
* Fast navigation without login requirements
* Real-world user experience similar to modern travel platforms
